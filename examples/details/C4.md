## Level 4 — Code (ParkService — key structural issues)

```mermaid
classDiagram
    class IParkService {
        <<interface>>
        +AddZone(name, isOpen)
        +AddDinosaurToZone(zoneName, dinosaur)
        +MoveDinosaur(from, to, name)
        +ToggleZone(zoneName)
        +CanSpeciesCoexist(s1, s2) bool
        +GetDinosaursInZone(zoneName) IEnumerable~Dinosaur~
        +GetAllZones() IEnumerable~Zone~
    }

    class ParkService {
        +ParkService()
        +AddZone(name, isOpen)
        +AddDinosaurToZone(zoneName, dinosaur)
        +MoveDinosaur(from, to, name)
        +ToggleZone(zoneName)
        +CanSpeciesCoexist(s1, s2) bool
        +GetDinosaursInZone(zoneName) IEnumerable~Dinosaur~
        +GetAllZones() IEnumerable~Zone~
        -Init()
    }

    class DataAccessLayer {
        <<static>>
        +$ _db : Database
        +$ Init()
        +$ Init(db)
        +$ SaveZone(zone)
        +$ GetZone(code)
        +$ SaveDinosaur(zoneCode, entity)
        +$ GetDinosaurs(zoneCode)
        +$ GetDinosaurByName(name)
    }

    class Database {
        +Zones : Dictionary~string,ZoneEntity~
        -_dinosaurs : Dictionary~string,DinosaurEntity~
        +$ Dinosaurs(db) Dictionary~string,DinosaurEntity~
    }

    class ReflectionHelper {
        <<static>>
        +$ GetPrivateField(obj, fieldName) object
    }

    class Dinosaur {
        +Name : string
        +Species : string
        +IsCarnivorous : bool
        +IsSick : bool
        +LastFed : DateTime
    }

    class Zone {
        +Name : string
        +IsOpen : bool
        +Dinosaurs : List~Dinosaur~
        +Status : string
    }

    class DinosaurEntity {
        +CodeName : string
        +Species : string
        +IsVegan : bool
        +HealthStatus : string?
        +FeedingTime : DateTime
    }
    class ZoneEntity {
        +ZoneCode : string
        +AccessStatus : bool
        +DinosaurCodes : List~string~
    }

    ParkService ..|> IParkService
    ParkService ..> DataAccessLayer : calls static methods
    DataAccessLayer ..> Database : mutates _db directly
    Database ..> ReflectionHelper : uses to expose _dinosaurs
    ParkService ..> Dinosaur : creates/reads
    ParkService ..> Zone : creates/reads
    DataAccessLayer ..> Entities : uses as persistence model

    %% NOTE: coupling is structural — ParkService CANNOT be tested without DataAccessLayer static state
```

> **Key structural problem:** `Database` deliberately makes `_dinosaurs` private, then immediately undermines encapsulation by calling `ReflectionHelper.GetPrivateField()` on itself. This is the core anti-pattern: the class actively breaks its own contract.
